## Java 篇

### Map

声明：`Map<T, T> map = new HashMap<T, T>();`

注意：当不指定泛型时，Map的value默认是一个Object。Map<String, Integer>

### List

声明：`List<T> list = new List<T>();`

### Array

声明：`char[][] map = char[3][3];`
